# 48in48 Nonprofit Website Projects

**Role:** Volunteer Project Manager
**Organization:** [48in48](https://48in48.org)
**Timeline:** 2023 - Present

---

## Overview

48in48 builds 48 nonprofit websites in 48 hours, pairing volunteers with nonprofits in need of a digital presence. As a Project Manager, I lead cross-functional teams of designers, developers, and content writers through rapid Agile sprints to deliver fully functional websites on time.

**Key Results:**
- 7 website launches completed with 100% on-time delivery
- Managed teams of 4-8 volunteers per sprint
- Improved nonprofit digital visibility by 15% on average post-launch
- Maintained RAID logs throughout each sprint to track risks and dependencies

## Project Folder Structure

Each project folder contains:

```
project-name/
├── project-charter.md       # Scope, objectives, stakeholders
├── raid-log.md              # Risks, Actions, Issues, Dependencies
├── kickoff-agenda.md        # Team kickoff meeting agenda
├── status-updates/          # Sprint progress updates
├── retrospective.md         # Post-project lessons learned
└── final-summary.md         # Outcomes and impact
```

---

## PM Approach

- **Methodology:** Agile/Scrum with 48-hour sprint structure
- **Stakeholder Management:** Nonprofit client briefings, volunteer team check-ins, and sponsor updates
- **Risk Management:** Active RAID log maintained throughout each sprint
- **Tools Used:** Jira, Google Workspace, Slack
